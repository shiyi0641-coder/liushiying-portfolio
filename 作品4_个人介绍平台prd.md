# 产品需求文档（PRD）

## 个人介绍平台 | Personal Portfolio Website

---

## 一、项目概述

### 1.1 产品定位

柳诗颖个人介绍平台是一个展示 AI 产品经理 / 心理学硕士专业形象的综合性个人网站，集成了个人信息展示、作品集呈现、AI 智能对话交互三大核心功能，旨在为潜在雇主、合作方及同行提供一个全方位了解本人的窗口。

### 1.2 核心目标

- **专业形象塑造**：通过精心设计的莫兰迪色系 UI 传递沉稳、专业的个人气质
- **信息高效传递**：以结构化方式呈现教育背景、实习经历、核心项目与技能图谱
- **互动体验增强**：接入 AI 分身对话功能，提供 7×24 小时智能问答服务
- **品牌记忆强化**：通过摄影板块展现个人生活侧面，塑造立体人物形象

### 1.3 目标用户

| 用户类型 | 核心需求 |
|----------|----------|
| 招聘方 / HR | 快速了解候选人背景、能力与经历 |
| 合作伙伴 | 评估专业能力与项目经验 |
| 同行从业者 | 了解行业实践与交流学习 |
| 媒体 / 记者 | 获取人物背景信息 |

## 三、视觉规范

### 3.1 设计风格

- **风格定位**：简约现代、温柔专业
- **配色方案**：莫兰迪色系（Morandi Colors）

### 3.2 色彩体系

| 色彩名称 | 色值 | 用途 |
|----------|------|------|
| 灰绿色 Gray-Green | #8B9A8E | 主色调、导航、按钮渐变 |
| 灰蓝色 Gray-Blue | #7B8B9E | 辅色调、实习经历卡片 |
| 粉色 Pink | #C9A5A5 | 强调色、心理学相关 |
| 奶油色 Cream | #F5F3F0 | 背景色、分区过渡 |
| 鼠尾草色 Sage | #A8B5A0 | 点缀色、教育背景 |
| 薰衣草色 Lavender | #B5A8C9 | 点缀色、技能证书 |

### 3.3 字体规范

- **主字体**：Inter（Google Fonts）
- **备选字体**：-apple-system, BlinkMacSystemFont, sans-serif
- **字重**：300（Light）、400（Regular）、500（Medium）、600（Semibold）、700（Bold）

### 3.4 动效规范

| 动效类型 | 参数 | 说明 |
|----------|------|------|
| 渐入动画 | fade-in-up, 0.6s ease-out | 元素进入视口时触发 |
| 悬停缩放 | scale(1.02), 0.3s ease | 卡片悬停时轻微放大 |
| 阴影过渡 | box-shadow 变化, 0.3s ease | 卡片悬停阴影增强 |
| 按钮悬停 | translateY(-2px), 0.3s | 按钮上浮效果 |
| 图片悬停 | scale(1.1), 0.5s | 生活照片悬停放大 |

## 五、响应式设计

### 5.1 断点规范

| 断点 | 屏幕宽度 | 适配内容 |
|------|----------|----------|
| 移动端 | < 768px | 单列布局、汉堡菜单 |
| 平板端 | 768px ~ | 双列网格 |
| 桌面端 | ≥ 1024px | 四列网格、展开导航 |

### 5.2 适配要求

- 导航栏：桌面端横排展开，移动端折叠为汉堡菜单
- Hero 区域：移动端上下堆叠，桌面端左右分栏
- 网格布局：自动响应列数调整
- 图片：使用 object-fit 保证填充效果
